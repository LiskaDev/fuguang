# 扶光AI助手 - Gradio Web UI 使用说明

## 🚀 快速启动

### 1. 安装依赖

```bash
pip install gradio pandas tabulate
```

### 2. 启动Web界面

```bash
python webui.py
```

### 3. 访问界面

浏览器打开：**http://localhost:7860**

## 🐛 常见问题

### Q: 启动失败，提示找不到模块
A: 请确保已安装所有依赖：`pip install -r requirements.txt`

### Q: 端口7860被占用
A: 修改webui.py中的`server_port=7860`为其他端口（如7861）

### Q: 性能监控没有数据
A: 需要先进行对话，系统才会记录性能数据

### Q: 如何生成公网链接？
A: 将`share=False`改为`share=True`（需要gradio账号）
